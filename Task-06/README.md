# Encryption & Cryptography – Mini Guide

## Objective
To understand basic cryptography concepts including symmetric and asymmetric encryption, AES file encryption, RSA key generation, digital signatures, hashing, and real-world applications.

## Topics Covered
- Symmetric vs Asymmetric Encryption  
- AES File Encryption  
- RSA Key Generation  
- Digital Signatures  
- Hashing & Integrity Verification  
- Encryption Algorithm Comparison  
- Real-world Usage (HTTPS, VPN)

## Key Takeaways
- AES is fast and used for data encryption  
- RSA is used for secure key exchange  
- Hashing ensures data integrity  
- Encryption is essential for secure communication
